# Outfront Studios Site v1

Premium marketing site for **Outfront Studios** built with **Vite + React**.

## Stack

- Vite
- React
- Clean custom CSS (no UI framework)

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in terminal (typically `http://localhost:5173`).

## Production build

```bash
npm run build
npm run preview
```

The build output is generated in `dist/`.

## Content included

- Hero with primary + secondary CTAs
- Positioning bar
- Three service pillars
- Three package tiers with pricing
- Optional add-ons
- Korea House v2 proof highlight
- Who this is for
- Final CTA + contact email

## SEO basics

Defined in `index.html`:

- Title + meta description
- Open Graph tags
- Twitter card tags
- Canonical URL
